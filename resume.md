### **Управление светодиодными лентами через MQTT**
*2023 — настоящее время*

**Описание:** Система управления 180 светодиодами в реальном времени через MQTT протокол.

**Реализация:**
- **Backend:** Spring Boot микросервис для управления устройствами
- **Протокол:** MQTT с QoS 1 для гарантированной доставки
- **Клиент:** Python-скрипты для генерации анимаций (30 FPS)
- **Устройство:** ESP8266 (Arduino) с адресной светодиодной лентой WS2812
- **Оптимизация:** Достигнута задержка менее 50ms на обработку кадра

**Технологии:** Java, Spring Boot, MQTT (Mosquitto), Python, Arduino, ESP8266, WebSocket
